# OMEGA Architecture

OMEGA is an AI Trading Operating System built around one core feedback loop.

```text
Knowledge
↓
Market Intelligence
↓
AI
↓
SignalFlow
↓
Paper Trading
↓
Analytics
↓
Learning
↓
Knowledge
```

## Core Rule

Every future feature must strengthen this loop. If a feature does not make the loop smarter, safer, more explainable, or easier to validate, it should not be added to the core platform.

## Operating Principles

- Mock-first.
- Interface-first.
- Provider-independent.
- TradingView-assisted, never TradingView-dependent.
- SignalFlow-driven.
- Paper-trading validated.
- Analytics-informed.
- Knowledge-enhanced.
- No autonomous live trading.

## Runtime Position

OMEGA v1.0 RC is a stable frontend and contract platform. It does not include live market feeds, broker APIs, exchange APIs, production databases, authentication, background workers, or real financial transactions.
